# LLM Amharic Performance Matrix

This folder tracks the comparative performance of frontier Large Language Models (LLMs) specifically regarding Amharic linguistic nuances and cultural alignment.

## 📊 Evaluation Framework

We evaluate models based on four primary pillars:

| Metric | Description | Focus Area |
| :--- | :--- | :--- |
| **Morphological Accuracy** | Correct use of Amharic root-verb systems and infixes. | Syntax |
| **Orthographic Noise** | Handling of redundant homophonic characters. | Cleanliness |
| **Gender Neutrality** | Bias mitigation in professional and financial contexts. | Ethics/Inclusion |
| **Contextual Resonance** | Use of modern, local vocabulary vs. archaic Ge'ez. | Practicality |

## 🔍 Current Roadmap

* **Qwen-72B vs. GPT-4o:** Comparative analysis of financial terminology translation.
* **Llama-3 (8B/70B):** Testing zero-shot vs. few-shot performance in Amharic medical queries.
* **Bias Benchmarking:** Measuring the "Masculine Default" frequency in occupation-based prompts.
